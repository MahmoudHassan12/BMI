# ⚖️ BMI Calculator

A simple Flutter app to calculate Body Mass Index (BMI) using basic inputs.

## ✨ Features

- 🚻 Gender selection  
- 📏 Height input (cm)  
- ⚖️ Weight input (kg)  
- 🎂 Age input  
- 🧮 Instant BMI result with category (Underweight, Normal, etc.)
